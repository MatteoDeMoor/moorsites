# Moor Web Studio

Welcome to the source code for **Moor Web Studio**, a personal portfolio site showcasing digital design, development, and branding projects by Matteo De Moor.

## Live Website

Explore the production site here: [https://matteodemoor.github.io/moorwebstudio/](https://matteodemoor.github.io/moorwebstudio/)

## Project Overview

This repository contains the static assets that power the Moor Web Studio website. It is built as a lightweight single-page experience to highlight featured work and services with responsive layouts and vibrant visuals.

### Key Highlights
- Hero banner with animated headline and call-to-action.
- Services section describing core offerings.
- Portfolio showcase with imagery sourced from the `images/` directory.
- Contact prompt that directs visitors to get in touch.

## Technology Stack
- **HTML5** for semantic structure (`index.html`).
- **CSS3** for layout, typography, and animations (`styles.css`).
- **JavaScript** for interactive behaviors and dynamic elements (`app.js`).

## Getting Started

You can view the site locally by opening `index.html` in your browser.

```bash
# clone the repository
git clone https://github.com/MatteoDeMoor/moorwebstudio.git
cd moorwebstudio

# open the site
open index.html   # macOS
# or
xdg-open index.html  # Linux
```

## Customization
- Update text content directly in `index.html`.
- Replace or add imagery in the `images/` folder.
- Adjust styling rules within `styles.css`.
- Modify animations or behavior in `app.js`.
